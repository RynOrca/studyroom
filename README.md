# 云端自习室

一个为学生和自学者打造的**多人在线实时专注学习平台**，融合了番茄钟计时、任务管理、实时互动和轻量笔记功能，让你在云端也能拥有沉浸式的学习体验。

---

## 项目特性

- **多人在线自习室**：加入自习房间，与学友一起实时专注，告别独自学习的孤独感。
- **智能专注计时**：支持经典番茄钟（专注+休息）和正计时模式，自定义时长，自动记录每一次专注块。
- **任务管理系统**：创建任务清单，设置今日目标，让学习计划清晰可见，专注更有方向。
- **实时互动功能**：支持开启麦克风、摄像头、共享屏幕，在互动区聊天交流，营造真实的自习室氛围。
- **专注数据统计**：自动生成每日、每月专注记录，通过日历直观展示你的学习投入，见证成长轨迹。
- **轻量笔记 (Memos)**：集成快速笔记功能，随时记录灵感、整理知识点，支持标签、待办和链接管理。
